Welcome to the PullApprove docs!

Frankly, we're proud of the fact that there isn't much documentation required to use PullApprove. But there are some things, and more on the way, that will need a small amount of documentation.

The majority of documentation is related to the .pullapprove.yml file, and pages for each of the available settings can be found in the menu on the left side of every page.

If you have any questions, comments, or ideas for PullApprove or these docs check out our public issues repo on GitHub!
